# Garbage-Collector

SpriteKit Game like "cut the rope". Based on raywenderlich tutorial. 

Slice the cable and choose, change, collect garbage, solve the puzzles, in this wonderful physics based game.

Features:

realistic physics
amazing graphics
amazing music
